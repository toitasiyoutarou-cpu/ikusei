CHEM GROW（高校化学 学習育成アプリ・問題追加版）

■ 起動方法
1. ZIPを展開します。
2. index.htmlをブラウザで開きます。
   GitHub Pages等にそのまま配置しても動作します。
3. 外部通信・外部ライブラリは使用していません。

■ 実装済み
- 主人公5人・ペット7匹の選択と途中変更
- 選択中の主人公・ペットが個別に成長
- 化学基礎・化学の大単元／小単元の土台
- 小単元ごとの独自能力値と経験値
- 複数小単元へ経験値を分配する問題設計
- 一問一答（単一選択・複数選択・数値・短答）
- 記述問題（模範解答＋観点別自己採点）
- 5問以内の学習セッションと結果表示
- 端末内のローカル保存
- セーブデータの書き出し／読み込み
- 問題未登録単元の「工事中」表示
- 01「物質の成分と構成元素」02「原子の構成と元素の周期表」03「化学結合」を中心に約438問を収録
- 学習到達度による単元マスター称号

■ 問題を追加する場所
data/questions.js

問題の基本形式：
{
  id: "一意のID",
  type: "single",
  difficulty: 1,
  baseXp: 8,
  prompt: "問題文",
  choices: ["選択肢1", "選択肢2", "選択肢3", "選択肢4"],
  answer: 0,
  explanation: "解説",
  unitWeights: [
    { id: "basic_mol", weight: 0.6 },
    { id: "basic_molar_mass", weight: 0.4 }
  ]
}

■ 問題形式
single   : 単一選択
multiple : 複数選択（answers: [0,2]）
numeric  : 数値入力（answer, tolerance, unitLabel）
short    : 短答（acceptedAnswers）
written  : 記述（modelAnswer, criteria）

■ セーブについて
localStorageを利用しています。ブラウザの履歴・サイトデータを削除すると消えるため、
設定画面からJSONのバックアップを定期的に書き出してください。

■ 校内限定について
このZIP自体には通信機能・認証機能はありません。
校内限定公開にする場合は、学校アカウント限定の配信環境や校内サーバー側でアクセス制御してください。


■ 今回追加した問題範囲
- 物質の分類、分離・精製、元素・同素体、炎色反応、元素検出、三態、熱運動
- 原子の構造、同位体、放射性同位体、電子配置、周期表、イオン
- イオン結合、電解質、共有結合、分子の形、極性、分子間力、金属結合、結晶

※問題は5問ずつランダムに出題されます。問題数が多い小単元は繰り返すたびに異なる問題が出やすくなります。
